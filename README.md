# exment-qr-managed-plugin
ExmentでのQRを使った備品管理のテストプラグイン

Exmentで管理番号を設定したテーブルで貸出と返却の機能を追加します。

現在は読み取りだけです。

コードも散らばっているため整理と修正が必要です。
